Tata Cara Membuat Website Sederhana

Langkah 1: alat Program

menginstal program editor kode seperti visual studio dan buat folder dalam file manager

Klik gambar untuk Download VS Code

Langkah 2: Tulis nama file yang ingin dibuat dan struktur awalan

setelah buat file buat file namafile.html yang diakhiri dengan .html sehingga bukan catatan, masukkan struktur HTML minimal (`html`, `head`, `body`) atau html:5(akan langsung membuat struktur).

index.html

pengenalan diri

Langkah 3: CSS

Buat file namafile.css sesuai dengan file dan hubungkan ke HTML Anda dengan link rel="stylesheet" href="namafile.css" . dengan ini kita bisa membuat desain dari web kita seperti font warna dan latar belakang.

body { background: #333; }

Langkah 4: Uji Coba

donload live server terlebih dahulu untuk memulai web serve dan pencet go live.Web akan keluar
