Mengenal Struktur Web

HTML:

HTML atau kepanjanganya HyperText Markup Language adalah kerangka yang mendefinisikan struktur dan konten situs web Seperti tulang pada manusia.HTML berfungsi untuk menyusun struktur dan konten halaman, seperti teks, gambar, tautan, tabel, dan elemen lainnya

contoh: link rel="stylesheet" href="" untuk menaruh link dan img src="" alt="" untuk mengambil gambar (hint: gambar harus dimasukan dalam satu folder)

CSS:

CSS atauCascading Style Sheet merupakan bagian pengatur visual atau design dari web. Ini mengatur warna, tata letak, font.seperti bagian baju dari web

h1 { color: blue; font-size: 24px; }

JavaScript:

JavaScript (JS) menambahkan fungsionalitas dan interaktivitas pada halaman web. Dengan JavaScript, sebuah website tidak hanya menampilkan informasi secara statis, tetapi juga dapat merespons tindakan pengguna, seperti menampilkan pesan ketika tombol ditekan.

let pesan = document.getElementById("pesan").value;

Tailwind:

Tailwind adalah framework utility-first untuk css. yang memberikan kelas siap pakai karena semua utilitas sudah disediakan. Hal ini membuat proses pengembangan web menjadi lebih cepat dan efisien.

button class="bg-blue-500 text-white px-4 py-2 rounded"
